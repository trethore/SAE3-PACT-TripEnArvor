# SAE3-PACT-TripEnArvor
**Tutoriel : Comment se connecter au projet**

1. Aller sur VS Code, 3ème onglet, puis cliquer sur le 1er bouton.
2. Cliquer sur les trois petits points, puis sélectionner "Clone". Coller l'URL suivante : https://github.com/trethore/SAE3-PACT-TripEnArvor.git
3. Placer le projet dans votre dossier de travail.

Dans le terminal :

```bash
git config --global user.name "<nomGithub>"
git config --global user.email "<emailGithub>"
```

**Pour changer de branche** :

1. Exécuter la commande suivante pour mettre à jour votre dépôt local :
   ```bash
   git pull
   ```
2. En bas à droite de VS Code, vous pouvez changer de branche en sélectionnant la branche souhaitée.
